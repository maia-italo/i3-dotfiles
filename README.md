# i3-dotfiles
My dotfiles for i3wm

### Dependencies

- autotiling
- playerctl
- zscroll
- picom
- dunst
- polybar
- rofi

```bash
$ sudo pacman -S autotiling spotify-laucher playerctl picom polybar libsysprof-capture alsa-card-profiles rofi-lbonn-wayland-git
$ yay -S zscroll-git
```
